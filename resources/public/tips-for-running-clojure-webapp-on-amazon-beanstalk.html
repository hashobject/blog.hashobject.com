Tips for running Clojure web app on Amazon Beanstalk

Logging

If you are using clojure.tools.logging in the app for logging following snippet will help you to have proper logs on Amazon Beanstalk.


  (alter-var-root
   #'clojure.tools.logging/*logger-factory*
   (constantly (clojure.tools.logging.impl/jul-factory)))

For Ring/Compojure projects just put this code somewhere on top of your web handler file.

Force HTTPS

I described how to get HTTPS support for Clojure web app deployed on Amazon Beanstalk in the previous post.

Sometimes you need to force HTTPS traffic for eevrything. So e.x. if user request page on http://example.com you want to redirect him on https://example.com.

You can setup proxy to do that.

I was able to achive that with these sample ring middleware:


  (defn https-url [request-url]
    (str (str (str (str "https://" (:server-name request-url) ":") "443")) (:uri request-url)))


  (defn require-https
    [handler]
    (fn [request]
      (if (= (:scheme request) :http)
        (ring.util.response/redirect (https-url request))
        (handler request))))

And then just add require-https middleware to your other middleware. E.x.:


  (def war-handler
    (-> app
      (require-https)
      (wrap-resource "public")
      (wrap-base-url)
      (wrap-file-info)))
